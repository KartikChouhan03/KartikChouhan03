# About Me

I'm **Kartik Chouhan**, a **B.Tech Information Technology** student at **Manipal University Jaipur** with a strong interest in building scalable backend systems, intelligent software, and AI-powered applications.

My work spans **Backend Development**, **Full Stack Engineering**, **Machine Learning**, **Computer Vision**, and **Natural Language Processing**, where I enjoy transforming complex real-world problems into practical software solutions.

Over the past few years, I've built projects involving **Automatic Number Plate Recognition (ANPR)**, **IoT-integrated Smart Parking Systems**, **Fake Review Detection using Transformer Models**, **Peer-to-Peer Rental Platforms**, and **Full-Stack Web Applications**.

I enjoy designing software that goes beyond user interfaces by focusing on clean architecture, REST APIs, modular systems, automation, and maintainable code.
