# 🚗 SmartParkX

### AI-Powered Smart Parking Infrastructure

> An end-to-end intelligent parking management platform that combines **Computer Vision**, **IoT**, and **Backend Engineering** to automate vehicle entry, parking allocation, billing, and exit management with minimal human intervention.

<br>

## System Workflow

```text
                Vehicle Arrival
                       │
                       ▼
          ANPR Camera (ESP32-CAM)
                       │
                       ▼
           YOLOv8 Plate Detection
                       │
                       ▼
          OCR Text Recognition
                       │
                       ▼
          Django REST Backend APIs
                       │
        ┌──────────────┼──────────────┐
        ▼              ▼              ▼
 Parking Engine   Slot Manager   Billing Engine
        │              │              │
        └──────────────┼──────────────┘
                       ▼
             PostgreSQL Database
                       │
                       ▼
              React Dashboard UI
```

---

## Engineering Overview

SmartParkX is designed as an intelligent parking infrastructure platform capable of automating the complete parking lifecycle. Instead of functioning as a traditional parking application, the system integrates computer vision, IoT devices, backend APIs, and a modern dashboard into a unified workflow capable of handling real-time vehicle identification, parking management, and billing.

The architecture emphasizes modularity, scalability, and automation, allowing each subsystem—vision, backend, sensors, billing, and frontend—to operate independently while communicating through REST APIs.

---

## Engineering Highlights

| Engineering Area | Implementation |
|------------------|----------------|
| 🧠 Computer Vision | Automatic Number Plate Recognition using YOLOv8 & OpenCV |
| ⚙ Backend | Django REST Framework powering parking sessions, users, vehicles, and payments |
| 📡 IoT Integration | ESP32-CAM with parking slot sensors for real-time monitoring |
| 🚘 Vehicle Tracking | Automated entry and exit recognition through ANPR |
| 💳 Billing Engine | Parking duration calculation with automated billing workflow |
| 📊 Dashboard | React-based interface for monitoring parking activity and slot availability |
| 🗄 Database | PostgreSQL for structured parking records and transaction history |
| 🔐 Security | JWT Authentication and Role-Based Access Control |

---

## Why This Project Matters

Unlike traditional parking management software, SmartParkX combines **Artificial Intelligence**, **Computer Vision**, **IoT**, and **Backend Engineering** into a single integrated platform capable of automating real-world parking operations.

The project demonstrates how multiple engineering domains can work together to create scalable intelligent infrastructure rather than isolated software modules.

# 🛡️ DeceptiScan

### AI-Powered Fake Review Detection Platform

> A production-oriented browser extension that combines **Natural Language Processing**, **Transformer Models**, and **FastAPI** to detect fake, AI-generated, and manipulative product reviews while generating explainable trust scores in real time.

<br>

## AI Inference Pipeline

```text
          Amazon Product Page
                   │
                   ▼
        Browser Extension (React)
                   │
                   ▼
      Intelligent Review Scraper
                   │
                   ▼
      Data Cleaning & Preprocessing
                   │
                   ▼
     Fine-Tuned DeBERTa-v3 Model
                   │
                   ▼
        Trust Score Generation
                   │
                   ▼
      Review Classification API
                   │
                   ▼
      Interactive Analytics Dashboard
```

---

## AI Engineering Overview

Unlike traditional sentiment analysis projects, DeceptiScan focuses on identifying deceptive and AI-generated reviews through a transformer-based inference pipeline.

The project combines browser automation, asynchronous scraping, backend inference services, and explainable AI to provide actionable credibility scores instead of simple positive or negative sentiment predictions.

---

## Model Performance

| Metric | Result |
|---------|--------|
| 📚 Dataset | **72,877** labeled reviews |
| 🧠 Model | Fine-Tuned **DeBERTa-v3** |
| 🎯 Accuracy | **96.8%** |
| ⚡ Latency | **~12 ms / Batch** |
| 🌐 Deployment | FastAPI Inference Server |
